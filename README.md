# A product Carousel Widget

A widget that displayes a product carousel, seen in real word systems like Airbnb, Amazon etc.

## Getting Started

Clone this project
`git clone https://github.com/Goketech/widget-presentation.git`

Navigate to project folder
`cd widget-presentation`

Run the project
`flutter run`

## Properties of the widget

- **autoScrollDuration**: Controls how long between each automatic page transition.

- **snapDuration**: Controls the animation speed of carousel transitions and page snapping.

- **viewportFraction**: Controls what fraction of the screen each card occupies.
